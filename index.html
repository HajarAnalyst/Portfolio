Data Visualisation Project
with PowerBI

Context & Objective
This project is based on a public dataset from a survey about data-related professions. The dataset was provided in raw format and required data extraction and cleaning before analysis. The objective is to analyze salaries, job satisfaction, country distribution, and programming languages used, in order to identify key trends in the data industry.

Data & Tools
Survey data collected from approximately 630 respondents.
Tools used: Power BI, Excel.

Key Analyses
• Average salary by job title
• Job satisfaction and work-life balance
• Popular programming languages
• Perceived difficulty of entering data-related roles

Deliverables
Interactive Power BI dashboard enabling filtering by job title, country, and survey responses.
