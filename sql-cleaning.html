Data Cleaning Project (SQL)

Case study showing how I clean and standardize a raw dataset using SQL.

Context & Objective

The goal of this project is to demonstrate a complete data-cleaning workflow using SQL: detecting inconsistencies, removing duplicates, standardizing formats, handling missing values, and preparing a clean dataset for analysis and reporting.

Data & Tools

  • Dataset: layoffs / workforce reduction dataset (raw format)
  • Tools: MySQL (SQL)

Cleaning Steps (high-level)

  • Remove duplicates (using staging tables / CTEs)
  • Standardize company names, industries, and country formats
  • Convert and validate date formats
  • Handle null / blank values and inconsistent fields
  • Prepare the cleaned table for further analysis

Deliverables

  • SQL script documenting the full cleaning workflow
  • Cleaned dataset/table ready for analysis

Results

  • Standardized columns (company / country / industry)
  • Duplicates removed and missing values handled
  • Clean table ready for analysis and visualization
